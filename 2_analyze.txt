Анализ страницы https://ru.wikipedia.org/

Главная страница https://ru.wikipedia.org/ состоит из сладующих блоков:
 - шапки, включающей в себя блоки авторизации и поиска
 - меню
 - поиском по сайту
 - основной части представляющей собой набор сниппетов
 - списка ссылок на проекты семейства Wiki
 - футера со ссылками на документы и описания самого проекта Wikipedia
 
 
 В шапке сайта располагается блок авторизации на проекте, с возможностью создать новую очетную запись.
 
 Под шапкой располагаем поисковую строку по статьям проекта.
 
 Левый верхний угол - стандартно, лого проекта. Под лого располагаем меню, состоящее из нескольких тематических блоков, 
 в каждом из которых есть ссылки на несколько подразделов. Меню не сворачивается и присутствует в раскрытом виде не каждой странице сайта. Здесь же размещается список
 языков статей для выбора пользователя.
 
 Основная часть спредставлена несколькими сниппетами (например, самая читаемая статья по итогам прошлого дня, выбор редакции, список событий текущего дня, новости и тд)
 с кнопками перехода в соотвествующие разделы, где информация будет представлена уже в более полном виде.
 
 Ниже сниппетов расположен блок со списком проектов семейства Wiki. Упоминание каждого проекта сопровождается лого проекта, названием и небольшим описанием, 
 чем проект может быть полезен (1-2 слова).
 
 В футере располагаем ссылки на служебные материалы (о проекте, политика конфиденциальности, обратная свзь и тп)
